---
title: Listar tags
api: "GET /api/v1/tags"
description: Lista todas as etiquetas (tags) do workspace, ordenadas por data de criação.
---

Uma **tag** é uma etiqueta colorida do workspace, aplicável a contatos.

## Autenticação

<ParamField header="Authorization" type="string" required>
  `Bearer vg_live_<sua_chave>` — API Key com escopo `tags:read`
</ParamField>

Retorna todas as tags do workspace, ordenadas por data de criação. Aceita `limit` (1–100, padrão 25) e `offset`.

<RequestExample>
```bash cURL
curl "https://crm.vistum.com.br/api/v1/tags?limit=50" \
  -H "Authorization: Bearer vg_live_SUA_CHAVE_AQUI"
```
</RequestExample>

<ResponseExample>
```json 200 — OK
{
  "data": [
    { "id": "tag_quente01", "name": "lead-quente", "color": "#22c55e", "createdAt": "2026-06-01T10:00:00.000Z" }
  ],
  "pagination": { "total": 1, "limit": 50, "offset": 0, "hasMore": false }
}
```
</ResponseExample>
