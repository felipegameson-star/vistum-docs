---
title: Excluir produto
api: "DELETE /api/v1/products/{id}"
description: Exclui um produto do catálogo do workspace.
---

## Autenticação

<ParamField header="Authorization" type="string" required>
  `Bearer vg_live_<sua_chave>` — API Key com escopo `products:write`
</ParamField>

Retorna `{ data: { id, deleted: true } }`. Rate limit: perfil de exclusão.

<RequestExample>
```bash cURL
curl -X DELETE https://crm.vistum.com.br/api/v1/products/prod_x1 \
  -H "Authorization: Bearer vg_live_SUA_CHAVE_AQUI"
```
</RequestExample>

<ResponseExample>
```json 200 — OK
{ "data": { "id": "prod_x1", "deleted": true } }
```
</ResponseExample>
